ESP32 Fingerprint Attendance System

An IoT-based biometric attendance solution using Arduino ESP32 with Firebase backend

Project Overview

This project implements a fingerprint-based attendance system using ESP32 microcontroller, R305 fingerprint sensor, and Firebase real-time database for cloud storage. The system allows for:

  • Enrollment of new fingerprints
  • Verification against stored templates
  • Real-time attendance logging to Firebase
  • Status indication via LEDs
  • Time synchronization via NTP

Hardware Components

  • ESP32 Development Board - Main controller
  • R305 Fingerprint Sensor - Biometric scanner

  • Libraries:

    • Firebase ESP32 Client

Key Features

Getting Started

  1. Assemble the hardware according to the wiring diagram
  2. Set up Firebase project and Realtime Database
  3. Configure Arduino IDE with required libraries
  4. Update WiFi and Firebase credentials in the code
  5. Upload the code to ESP32
  6. Enroll fingerprints using serial commands
  7. Start logging attendance

For detailed instructions, refer to the Setup Guide.
